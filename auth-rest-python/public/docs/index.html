Methods

  • Auth

    Autenticação

    • Administrador

      no implementation for this version
      login para administradores
      Request:
      Resource:
      http://localhost:8086/api/auth/auth/admin
      Body:
      {
      "email": string,
      "password": string
      }
      Response:
      Body:
      {
      "access_token": string,
      "user": User
      }
      Codes:
      Code Message Description
      200 OK
      400 Invalid params - email and password are required
      401 Email or password invalid
      403 You do not have administrator permission
      404 User not Found
      Sample:
      Request:
      POST /auth/admin HTTP/1.1 Content-Type: application/json; charset=utf-8 { "email": "my_email", "password": "my_password" }
      Response:
      HTTP/1.1 200 OK Content-Type: application/json; charset=utf-8 { "access_token": "my_access_token", "user": { "created_on": "2018/05/02", "email": "my_email", "func": 2, "id": 1, "name": "my_name", "photo": "my_photo", "updated_on": "2018/05/02" } }

    • Facebook

      no implementation for this version
      login via token (facebook)
      Request:
      Resource:
      http://localhost:8086/api/auth/auth/facebook/{token}
      URI Parameters:
      Parameter Type Optional Description
      token string Token pessoal enviado pelo facebook
      Response:
      Body:
      {
      "access_token": string,
      "user": User
      }
      Codes:
      Code Message Description
      201 Created
      403 Invalid OAuth access token
      Sample:
      Request:
      GET /auth/facebook/my_token HTTP/1.1 Content-Type: application/json; charset=utf-8
      Response:
      HTTP/1.1 200 OK Content-Type: application/json; charset=utf-8 { "access_token": "my_access_token", "user": { "created_on": "2018/05/02", "email": "my_email", "func": 2, "id": 1, "name": "my_name", "photo": "my_photo", "updated_on": "2018/05/02" } }

    • Validate

      no implementation for this version
      validação pelo token de usuário
      Request:
      Resource:
      http://localhost:8086/api/auth/auth/validate/{token}
      URI Parameters:
      Parameter Type Optional Description
      token string Token do usuário
      Response:
      Body:
      {
      "func": number,
      "id": number
      }
      Codes:
      Code Message Description
      200 OK
      403 Invalid OAuth access token
      Sample:
      Request:
      GET /auth/validate/my_token HTTP/1.1 Content-Type: application/json; charset=utf-8
      Response:
      HTTP/1.1 200 OK Content-Type: application/json; charset=utf-8 { "func": 2, "id": 1 }
